# 懒得动 · 手机网页高保真设计稿

## 页面

1. 首页：定位确认与单一开始入口
2. 偏好选择：三个必要条件，其他条件延后
3. 盲盒结果：单一推荐、两次免费重抽
4. 玩法详情：步骤、预算、时间与地图
5. 今日待办：从确认到实际出发的行动闭环

## 设计基线

- 画板尺寸：390 × 844 px
- 主色：`#7157FF`
- 背景：`#FFFAF2`
- 正文色：`#3E3B54`
- 强调色：`#FFD166`
- 最小主要按钮高度：58 px
- 主要圆角：19–29 px

## 查看单个页面

在地址后添加参数：

- `?screen=home`
- `?screen=preferences`
- `?screen=result`
- `?screen=detail`
- `?screen=todos`

不添加参数时显示完整设计看板。
